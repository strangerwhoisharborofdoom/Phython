# Phython - Basic Python Code for Beginners

## Overview
This repository contains a comprehensive collection of Python programs and examples designed for beginners. It covers fundamental programming concepts, operators, control structures, and practical programming exercises.

## Repository Structure

The repository is organized into three main categories:

### 1. Unit 1: Basic Operators (1a.py - 1j.py)
- **1a.py**: Input and Output program - Basic console I/O operations
- **1c.py**: Arithmetic operations - Mathematical operations (addition, subtraction, multiplication, division, modulus)
- **1d.py**: Relational and Logical operators - Comparison and boolean operations
- **1e.py**: Bitwise operators - Bitwise AND, OR, XOR, and shifts
- **1f.py**: Assignment operators - Variable assignment and compound operators
- **1g.py**: Membership check program for a list - Using 'in' operator with lists
- **1h.py**: Membership check program for strings - String membership operations
- **1i.py**: Identity operator demonstration - Using 'is' operator with integers
- **1j.py**: Comparison of list objects - List comparison operations

### 2. Unit 3: Control Structures and Loops (3a.py - 3i.py)
- **3a.py**: Check if a number is even or odd - If-else conditional statements
- **3b.py**: Number classification program - Multi-way conditionals
- **3c.py**: Find largest of three numbers - Nested conditional logic
- **3d.py**: While loop example - Loop control and iteration
- **3e.py**: For loop example - Iteration over sequences
- **3f.py**: Calculate sum of first N numbers - Loop-based calculations
- **3g.py**: Multiplication table script - Nested loops
- **3h.py**: Digit counting functionality - String and loop manipulation
- **3i.py**: Factorial calculation - Recursive and iterative approaches

### 3. Practice Programs (pr1.py - pr6.py)
- **pr1.py**: User input and discount calculation - Practical arithmetic with user input
- **pr2.py**: Area calculation and basic arithmetic operations - Geometric calculations
- **pr3.py**: Number comparison and grading system - Conditional logic and grading
- **pr4.py**: Multiplication table and count display - Multiple loop control
- **pr5.py**: Recursive factorial function - Function recursion
- **pr6.py**: Even/odd checker based on user input - Input validation and conditionals

## Getting Started

### Prerequisites
- Python 3.x installed on your system
- A text editor or IDE (VS Code, PyCharm, etc.)
- Basic understanding of command line/terminal

### Running the Programs

1. Clone the repository:
```bash
git clone https://github.com/strangerwhoisharborofdoom/Phython.git
cd Phython
```

2. Run any Python file:
```bash
python filename.py
```

For example:
```bash
python 1a.py
python 3a.py
python pr1.py
```

## Learning Path

**Beginner Level:**
1. Start with `1a.py` to understand input/output
2. Move through `1c.py` to `1j.py` to master operators
3. Practice with `3a.py` to `3c.py` for conditionals

**Intermediate Level:**
4. Learn loops with `3d.py` and `3e.py`
5. Explore functions and recursion with `3i.py` and `pr5.py`
6. Practice with remaining practice programs

## Tips for Learning

- Run each program and experiment with different inputs
- Modify the code to understand how changes affect output
- Try to write similar programs from scratch
- Use print statements to debug and understand program flow
- Comment your code to explain logic
